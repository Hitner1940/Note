### 第三部分：金屬的提取方法 (對應圖片 Page 2, 3)

提取方法的選擇取決於金屬在活性序中的位置：

1. **物理方法 (Physical Method)：**
    
    - 淘金 (Panning) 
        
        ```
        →→
        ```
        
         適用於金 (Au)。
        
2. **直接加熱 (Heating alone)：**
    
    - 適用於活性較低的金屬氧化物（如汞、銀）。
        
    - 方程式：
        
        ```
        2HgO→2Hg+O22HgO→2Hg+O2​
        ```
        
    - 方程式：
        
        ```
        2Ag2O→4Ag+O22Ag2​O→4Ag+O2​
        ```
        
3. **碳還原法 (Carbon Reduction)：** (重點)
    
    - 適用於活性中等的金屬（鋅、鐵、鉛、銅）。
        
    - 原理：碳 (C) 的活性比這些金屬高，能將氧從金屬氧化物中「搶」走。
        
    - **還原劑：** 碳 (C)、一氧化碳 (CO)、氫氣 (
        
        ```
        H2H2​
        ```
        
        )。
        
    - 方程式範例：
        
        - ```
            Fe2O3+3C→4Fe+3CO2Fe2​O3​+3C→4Fe+3CO2​
            ```
            
        - ```
            CuO+H2→Cu+H2OCuO+H2​→Cu+H2​O
            ```
            
4. **電解法 (Electrolysis)：**
    
    - 適用於活性極高的金屬（鉀、鈉、鈣、鎂、鋁）。
        
    - 範例：從鋁土礦提取鋁。
        
    - **關鍵點：** 需加入**冰晶石 (Cryolite)** 以降低氧化鋁的熔點（約 950°C），節省能源。

### 第四部分：實驗——利用碳提取金屬 (對應圖片 Page 5, 6)

這是考試常考的實驗題型。

1. **實驗目的：** 利用碳粉將金屬氧化物（氧化鐵、氧化銅、氧化鉛）還原。
    
2. **化學反應方程式 (必背)：**
    
    - **氧化銅 (黑色)** 
        
        **`→→`**
        
         **銅 (紅棕色)：**  
        
        ```
        2CuO+C→2Cu+CO22CuO+C→2Cu+CO2​
        ```
        
    - **氧化鉛 (黃色)** 
        
        **`→→`**
        
         **鉛 (銀灰色)：**  
        
        ```
        2PbO+C→2Pb+CO22PbO+C→2Pb+CO2​
        ```
        
    - **氧化鐵 (紅棕色)** 
        
        **`→→`**
        
         **鐵 (銀灰色)：**  
        
        ```
        2Fe2O3+3C→4Fe+3CO22Fe2​O3​+3C→4Fe+3CO2​
        ```
        
3. **實驗注意事項 (解釋題重點)：**
    
    - **Q: 為什麼加熱時要用碳粒覆蓋混合物？**
        
        - A: 防止空氣進入，確保還原反應進行，減少熱流失。
            
    - **Q: 為什麼加熱後需待混合物冷卻至室溫才打開坩堝蓋？**
        
        - A: 防止剛生成的熱金屬再次與空氣中的氧氣反應（防止再氧化）。
            
4. **結論：**
    
    - 碳是**還原劑 (Reducing agent)**。
        
    - 碳能將氧從金屬氧化物中移除。

### 第五部分：金屬的性質與用途 (對應圖片 Page 3, 4)

金屬的用途直接取決於其物理性質。

|   |   |   |
|---|---|---|
|金屬 (Metal)|用途 (Use)|選用的理由/性質 (Reason/Property)|
|**鋁 (Al)**|製造飛機機身|密度低（輕）、強度高|
||製造鋁罐|抗腐蝕、無毒、可延展|
|**銅 (Cu)**|製造電線|**導電性**極佳|
||製造煮食器具|**導熱性**極佳|
|**金 (Au)**|製造飾物|具光澤、不活潑（不易氧化變色）、延展性好|
|**鐵 (Fe)**|製造橋樑、建築|硬度高、強度高|
|**鉛 (Pb)**|製造魚鉤上的墜|**密度高**（重）|
|**鈦 (Ti)**|製造人工關節|質輕、強度高、與人體相容性好|

**金屬的一般物理性質 (Physical Properties)：**

1. 有光澤 (Lustrous)
    
2. 密度高 (High density)
    
3. 硬度高 (High hardness)
    
4. 延性及展性 (Ductile and Malleable)
    
5. 熔點及沸點高 (High melting/boiling point)
    
6. 良好的導熱及導電體 (Good conductor of heat/electricity)
