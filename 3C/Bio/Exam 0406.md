## Ch.12.1 食物的組成成份（已有知識）

- 原生質的主要元素：碳、氫、氧、氮等
    
- 碳水化合物的基本單位：單糖（葡萄糖），元素：C、H、O，H:O = 2:1
    
- 蛋白質的基本單位：氨基酸，元素：C、H、O、N、P、S 等
    
- 脂質的基本單位：脂肪酸 + 甘油，元素：C、H、O，H:O > 2:1
    
- 食物的五大功用：
    
    - 合成原生質（生長、修補）
        
    - 提供能量（呼吸作用）
        
    - 維持正常機能、預防疾病
        
    - 過多會肥胖、心臟病、高血壓
        
    - 不足會瘦弱、營養缺乏病、死亡

## Ch.12.2 營養與健康（主要食物物質）

### 碳水化合物

- 來源：米飯、麵包、薯仔、糖
    
- 功能：能量來源
    
- 缺乏：瘦弱、體力衰退
    
- 過多：肥胖
    

### 脂質

- 來源：豆類、花生、植物油、牛油、芝士、動物脂肪
    
- 功能：能量（39 kJ/g）、構成細胞膜、隔熱保暖、保護內臟、運送脂溶性維他命
    
- 缺乏：身體瘦弱
    
- 過多：貯存於皮下
    

### 蛋白質

- 來源：肉類、魚、豆類、蛋、豆腐、奶、芝士
    
- 必需氨基酸：8種需從膳食攝取
    
- 功能：原生質主要成份（生長修補）、製造酶/抗體/激素、能量（18 kJ/g）
    
- 缺乏：瘦弱、易感染疾病
    
- 過多：轉為尿素排出
    

### 維他命

- 維他命A（脂溶性）：魚肝油、蔬菜 → 夜盲症
    
- 維他命C（水溶性）：生果、蔬菜 → 壞血病
    
- 維他命D（脂溶性）：牛奶、芝士 → 佝僂病、軟骨病
    
- 特性：A/D過量會中毒；C過量隨尿排出；C不耐熱
    

### 礦物質

- 鈣：骨骼、牙齒、凝血 → 缺乏：佝僂病、軟骨病、不能凝血
    
- 鐵：血紅蛋白 → 缺乏：貧血
    
- 碘：甲狀腺素 → 缺乏：大頸泡（甲狀腺腫）
    

### 食用纖維

- 來源：蔬菜、生果
    
- 功能：刺激腸蠕動、防止便秘、幫助消化吸收
    
- 不能被人體消化吸收
    

### 水

- 來源：食水、飲品、食物、代謝水
    
- 功能：原生質主要成份、良好溶劑（介質、吸收、排泄、運輸）、參與水解、調節體溫（排汗冷卻）、潤滑劑（關節滑液）

## Ch.12.3 食物的消化和吸收

### 消化系統主要器官

- 口腔 → 食道 → 胃 → 小腸 → 大腸
    

### 機械消化

- 牙齒咀嚼（增加表面積）
    
- 胃攪動（形成糊狀）
    
- 膽汁乳化脂質（物理作用）
    

### 化學消化（酶）

- 唾液（澱粉酶）→ 口腔：澱粉 → 糖
    
- 胃液（胃蛋白酶）→ 胃：蛋白質 → 肽
    
- 胰液（多種酶）→ 小腸：分解澱粉、脂質、蛋白質
    
- 腸液 → 小腸：完成消化
    

### 膽汁

- 由肝製造，儲於膽囊，不含酶，呈鹼性
    
- 功能：乳化脂質（增加表面積）
    

### 吸收位置

- 小腸絨毛：
    
    - 水溶性物質（葡萄糖、氨基酸、礦物質、維他命C、水）→ 微血管（血液）
        
    - 脂溶性物質（脂質、維他命A、D）→ 乳糜管（淋巴）
        

### 大腸

- 吸收剩餘水分
    
- 形成糞便（含膽汁色素 → 棕色）

## Ch.12.4 均衡膳食

### 食物金字塔（由底至頂）

1. 穀物類（吃最多）→ 提供能量
    
2. 蔬菜、水果類（吃多些）→ 維他命、礦物質、食用纖維
    
3. 肉、魚、蛋、豆類（吃適量）→ 蛋白質（生長修補）
    
4. 奶類及代替品
    
5. 脂肪、油、鹽、糖（吃最少）
    

### 能量單位

- 千焦耳（kJ）、千卡路里（kcal）
    
- 1 kcal = 4.2 kJ
    

### 影響能量需求的因素

- 年齡：兒童/青少年需要較多
    
- 性別：男性一般較高（肌肉較發達）
    
- 活動量：體力勞動多需要更多
    

### 能量不均衡的後果

- 過量進食 → 肥胖、高血壓、心血管疾病
    
- 進食不足 → 瘦弱、營養缺乏病
    

### 食物標籤

- 列明成分、重量/體積、營養資料、能量值

## Ch.12.5 健康與疾病

### 傳染病

- 成因：微生物（細菌、真菌、原生動物）或病毒 → 病原體
    
- 傳播途徑：
    
    - 飛沫（咳嗽、打噴嚏）→ 戴口罩、洗手、通風
        
    - 皮膚傷口 → 包紮傷口、戴手套
        
    - 受污染食物/水/泥土 → 煮沸水、妥善處理食物
        
    - 蚊子叮咬 → 驅蚊劑、清除積水
        
- 預防：保持個人及環境衛生、接種疫苗
    

### 非傳染病

- 不會人傳人，與風險因素有關（生活模式、年老、遺傳）
    

### 心血管疾病（冠心病、中風）

- 風險因素：肥胖、高血壓、吸煙、高脂飲食、缺乏運動、食用纖維不足
    
- 成因：膽固醇積聚動脈內壁 → 管腔變窄 → 血流減少
    

### 癌症（肺癌、大腸癌）

- 肺癌與吸煙（焦油）密切相關
    
- 大腸癌風險因素：肥胖、進食過量紅肉/加工肉類、缺乏運動、吸煙、喝酒、食用纖維不足
    
- 其他致癌物：酒精、紫外光、乙型肝炎病毒
    

### 糖尿病

- 胰島素不足或身體無反應 → 血糖過高 → 尿糖
    
- 風險因素：肥胖、高血壓、缺乏運動
    

### 生物工程應用

- 基因重組大腸桿菌生產胰島素、生長激素
    
- PCR（聚合酶鏈反應）：檢測遺傳病、病毒（如COVID-19）
    
- 疫苗：注射後白血球記憶病原體，快速反應

## 補充：植物營養器官（根、莖、葉）— 工作紙 P.28-35

### 根

- 功能：吸收水分及礦物質、固定植物、貯藏食物、繁殖（營養繁殖）
    
- 縱切面構造：根冠（保護）、生長點（細胞分裂）、伸長區（細胞延長）、根毛區（增加表面積）、成熟區（分化）
    
- 橫切面：表皮、皮層、中柱（柱鞘、維管束、髓部）
    

### 莖

- 功能：支持、運輸（木質部運水/礦物質；韌皮部運有機食物）、貯藏、繁殖
    
- 外部特徵：有節、節間、芽、葉
    
- 橫切面：表皮、皮層、維管束環狀排列、髓部（貯藏）
    

### 葉

- 功能：光合作用、氣體交換（氣孔）、蒸騰作用、貯藏、繁殖
    
- 橫切面構造：
    
    - 角質層（保護、減少失水）
        
    - 上/下表皮（保護）
        
    - 柵狀組織（近葉面，多葉綠體 → 光合作用主場，葉面較深色）
        
    - 海綿組織（近葉底，細胞疏鬆 → 氣室，利氣體交換）
        
    - 葉脈（維管束）：木質部（上，運水）、韌皮部（下，運有機食物）
        
    - 氣孔（保衛細胞控制開關）
