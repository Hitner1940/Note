# 📚 中三中國語文科中段考 — 重點複習

## 一、讀本問題

### 單元三：自然禮讚 — 借景抒情

#### 講讀：《歸園田居》（其三） — 陶淵明

**（一）課文背景**

陶淵明是我國文學史上第一位田園詩人。他辭去彭澤令，回歸田園隱居，寫下《歸園田居》一組共五首詩，大約作於歸隱後第二年。本詩是組詩的第三首，描寫他親自參與農耕的真實生活體驗。

**（二）篇章內容**

種豆南山下，草盛豆苗稀。晨興理荒穢，帶月荷鋤歸。道狹草木長，夕露沾我衣；衣沾不足惜，但使願無違。

**（三）主旨思想**

1. **熱愛田園生活**：作者雖早出晚歸、辛勞耕作，但樂在其中，表現了對田園生活的投入和珍惜。
2. **追求精神自由**：「但使願無違」是詩的點睛之筆。陶淵明追求精神自由，不願為五斗米折腰，寧可歸隱田園。他滿足於簡樸恬淡的生活，認為心境的平靜比物慾權力更為可貴。

**（四）寫作手法 — 借景抒情**

- **白描手法**：語言樸素自然，不作雕琢修飾，直接描寫田間勞動情景。
- **間接抒情**：通過「帶月荷鋤歸」、「夕露沾我衣」等田園意象，寄託對隱逸生活的滿足和嚮往。詩中寫景，實為抒情。
- **直抒胸臆**：末句「衣沾不足惜，但使願無違」直接表露心跡，表明堅守高潔節操的決心。

**（五）感情分析**

從「晨興理荒穢，帶月荷鋤歸」可見作者雖日復一日辛勤勞動，卻全無怨言，反而帶着悠閒自得之情；「道狹草木長，夕露沾我衣」環境雖艱，他卻不以為苦，足見其對田園生活的熱愛。

> 📝 **考試提示**：
> - 若問「願」的含意：「願」是指陶淵明堅守高潔、遠離官場的精神追求。
> - 若問作者的生活態度：知足、安貧，追求內心平靜，不為物慾所困。


#### 講讀：《桃花源記》 — 陶淵明

**（一）課文背景**

此文是《桃花源詩》的序言，寫於東晉末年社會動盪之際。陶淵明目睹時局混亂、民不聊生，因而虛構出一個與現實社會截然對立的理想世界。

**（二）篇章結構 — 以「尋桃林—探桃源—辭桃源」為主線展開**

| 結構 | 內容 |
|------|------|
| 尋桃林 | 武陵漁人沿溪捕魚，偶然發現桃花林 |
| 探桃源 | 進入桃花源，見證其間美好生活 |
| 辭桃源 | 離開後再尋不得，「後遂無問津者」 |

**（三）桃花源的特點**

- **環境優美**：土地平曠，屋舍儼然，有良田美池桑竹之屬。
- **社會安寧**：阡陌交通，雞犬相聞。其中往來種作，男女衣着悉如外人。
- **生活幸福**：黃髮垂髫，並怡然自樂。
- **無階級剝削**：自給自足，人人平等，沒有官府的苛捐雜稅和戰亂之苦。

**（四）主旨思想**

1. **對現實社會的不滿與批判**：東晉末年政局動盪、戰亂頻仍，百姓生活困苦。陶淵明通過描寫桃花源的安寧美好，與混亂的現實形成強烈對比，表達了對現實社會的厭棄。

2. **對理想社會的嚮往**：虛構了一個沒有壓迫、沒有剝削、人民安居樂業的大同世界，寄託了作者的政治理想。

3. **對時代的絕望與深沉悲慨**：結尾「後遂無問津者」，既暗示桃花源的不存在，更蘊含陶淵明對時代的深層絕望 — 這樣美好的理想社會，終究無人能夠實現。

**（五）寫作手法**

- **虛實結合（虛景實寫）**：將虛構的桃花源通過細緻的景物描寫，寫得彷彿真有其地。這既有助讀者投入其中，亦使諷刺效果更為深刻。
- **漁人視角敘事**：以漁人的行蹤帶出所見所聞，客觀含蓄地呈現桃花源的美好。
- **今昔對比**：桃花源中人自云「先世避秦時亂」，與東晉末年的戰亂形成呼應。

> 📝 **考試提示**：
> - 若問「為什麼以『後遂無問津者』作結」：一方面暗示理想社會的虛幻，另一方面表達對現實的絕望，理想終究無法實現。

### 單元六：讀書與反思 — 議論方法（先破後立）

#### 講讀：《不求甚解》（節錄） — 馬南邨

**（一）課文結構 — 論說文三部分**

| 部分 | 段落 | 內容重點 |
|------|------|----------|
| 引論 | 第1段 | 對一般人盲目反對「不求甚解」的態度提出質疑 |
| 本論 | 第2-8段 | 闡述陶淵明「不求甚解」的本意及正確讀書態度；列舉諸葛亮、陸象山事例論證 |
| 結論 | 第9段 | 建議要反覆閱讀重要書籍，每讀一次都能開卷有益 |

**（二）「不求甚解」的兩種理解**

- **一般人的理解（錯誤）**：以為是讀書只求知道大概，不求徹底了解，對知識抱持一知半解的態度。【0†B】
- **作者的理解（正確）**：指領會書籍內容精髓和主旨，不在於死摳字句。這是一種強調整體把握、抓住核心的讀書態度。【0†D】

**（三）核心論點**

作者指出陶淵明「不求甚解」的真正意思包含 **「虛心學習」** 和 **「融會貫通」** 兩個層面。這並非馬虎了事，而是一種高明的讀書法：不拘泥於詞句細節，以免因小失大，而是抓住全書精神要旨。

**（四）論證方法 — 先破後立**

| 步驟 | 內容 |
|------|------|
| **先破（破斥錯誤觀點）** | 作者先批判一般人對「不求甚解」的錯誤理解（以為是一知半解、不認真讀書），指出這是對陶淵明的誤解 |
| **後立（建立正確觀點）** | 然後闡述「不求甚解」的真正含意，引用陶淵明的讀書態度加以正本清源 |

> **先破後立的優點**：通過一破一立、反覆論證，使自己的論點更加客觀有力；同時可以引起讀者興趣，增強文章說服力。

**（五）舉例論證 — 諸葛亮 vs 徐庶等人**

| 人物 | 讀書方法 | 態度 |
|------|----------|------|
| 諸葛亮 | 「觀其大略」 | 不死摳一字一句，注重整體把握；更高明 |
| 徐庶等人 | 「務於精熟」 | 精讀細究每一字句 |

**作者的觀點**：諸葛亮的方法更為優勝，因為掌握全書主旨遠比死記字句更為重要。讀書如果只顧局部而忽略整體，最終無法真正理解書中精義。

**（六）引用論證 — 陸象山的語錄**

陸象山指出讀書遇到不明白的地方應暫且跳過，不要拘泥固執於不明白之處，否則會因小失大。這與「不求甚解」的精神相通 — 讀書重在領會整體大義。

**（七）作者主張的讀書方法**

1. 讀書要領會著作要旨，抓住精神實質
2. 讀書要對著作有全面的了解，不能只顧局部
3. 重要的書必須反覆閱讀，因為許多書要多看才能真正讀懂其中的道理
4. 不應只記住經典的一些字句，而要理解精神實質

> ❌ **不是作者的主張**：「讀書要仔細推敲字裏行間的含義」 — 作者雖重視領會精神，但強調不應拘泥字句，不是反對仔細閱讀。

**（八）全文主旨**

作者指出「不求甚解」並非讀書馬虎，而是指讀書務求領會內容精髓、不在乎斟酌字句，這是古人的正確讀書方法，我們應該虛心學習。


### 單元七：借事說理

#### 導讀：《重返灣仔》 — 席慕容

**（一）篇章結構**

| 部分 | 段落 | 內容大意 |
|------|------|----------|
| 回憶舊居 | 第1-3段 | 回憶當年與家人搬進灣仔舊居的情況，交代地理位置和室內間隔 |
| 童年樂事 | 第4-9段 | 記述平台上有四個花池，回憶兒時與鄰居和弟妹玩樂、在花圃觀賞木芙蓉花的情景 — 抒發對童年的懷念 |
| 遷居與重訪 | 第10-12段 | 遷居台灣後多次重遊舊居，提到第一次和弟弟重返灣仔 |
| 尋訪舊居 | 第13-21段 | 記述與弟弟尋訪灣仔舊居的經過，描寫沿途景色的今昔對照 |
| 感懷 | 第19-22段 | 找到舊居位置，發現其改變，感慨時光流逝；為仍能找到街巷輪廓而感滿足和安慰 |

**（二）寫作手法**

**1. 借景抒情**

- 作者描寫童年時舊居附近的木芙蓉花等自然景色，流露對舊居和美好童年生活的懷念。
- **木芙蓉花的描寫**：從清晨的純白漸變為黃昏的桃紅，顏色變化細膩生動。作者把它比喻為雪、櫻和人喝酒後泛紅的臉，極具詩意。
- 這種細緻的景物描寫表達了對木芙蓉花的喜愛，也反映出童年時賞花的滿足之情。

**2. 步移法**

根據與弟弟尋訪舊居的路線，按順序描寫沿途見聞：從**修頓球場**開始，經過**昔日涼茶店的位置**，再走到**秀華台**。通過這種記敍方式，清晰展現舊居附近景色的今昔對照。

**3. 定點描寫**

在第21段，作者站在**秀華台的平台**上作為觀察點，隨着視線的轉移，描寫眼前花池的現況 — 花池已寸草不生，堆滿建材和雜物。通過定點觀察，集中呈現平台的破敗景象。

**4. 多感官描寫**

- **視覺**：描寫木芙蓉花的顏色變化。
- **聽覺**：描寫修頓球場「歡聲震耳」的情景。

**（三）情感分析**

| 情景 | 感情 |
|------|------|
| 木芙蓉花的細緻描寫 | 對木芙蓉花的喜愛、童年賞花的滿足之情 |
| 童年與鄰居和弟妹玩樂 | 對童年生活的懷念 |
| 秀華台的破敗景象 | 對物事變遷的感慨 |
| 仍能找到舊日街巷的輪廓 | 滿足、欣慰，認為這是「奇跡」般難能可貴 |

**（四）重點句子語譯**

| 句子 | 語譯 |
|------|------|
| 已而復然，又久而後安之 | 過了不久又再這樣，日子久了，我便習慣了。 |
| 足履平地，不與窪適也 | 腳習慣踏在平地，便不適應踏在窪地。 |

（以上兩句亦見於中三讀本，屬《習慣說》中的重要語譯部分。）

**（五）全文主旨**

作者通過回憶童年時在灣仔舊居的生活片段，以及多年後重訪灣仔所見的景物變化，抒發對舊居和愉快童年的懷念之情，也表達了對時光流逝、物事變遷的感慨。


#### 導讀：《兵車行》 — 杜甫

**（一）課文背景**

《兵車行》是唐代大詩人杜甫創作的敍事詩。當時是盛唐天寶年間（約8世紀中葉），唐玄宗連年對西北、西南少數民族發動戰爭，百姓被強行徵兵，民生困苦。杜甫眼看民間疾苦，憤而寫下這首詩。

> 🔍 **重要背景**：杜甫沒有沿用舊有的樂府古題，而是「緣事而發，即事名篇」，自創新題來寫時事，這對後世新樂府運動有開創性影響。

**（二）篇章結構**

全詩分為兩大段落：

| 段落 | 形式 | 內容重點 |
|------|------|----------|
| 第1段 | 紀事（客觀描寫） | 摹寫送別的慘狀，如「耶娘妻子走相送，塵埃不見咸陽橋。牽衣頓足攔道哭，哭聲直上干雲霄。」 |
| 第2段 | 紀言（借征夫之口訴說） | 借征夫對老人的答話，傾訴人民對戰爭的痛恨 |

**（三）主旨思想**

1. **反對戰爭**：通過描寫百姓被強徵入伍、親人送別的淒慘場面，表達對戰爭的強烈控訴。
2. **同情百姓**：征夫自述「或從十五北防河，便至四十西營田」，一生在戰役中度過，家庭被破壞。詩人對戰亂中的人民寄予深切同情。
3. **揭露統治者的窮兵黷武**：借征夫之口，揭露唐玄宗長期的對外用兵政策給民間帶來的巨大災難。

**（四）寫作手法**

- **借敍事抒情**：全詩沒有直接說理，而是將批判融入敍述之中。首段以聲畫渲染（車聲、馬鳴聲、哭聲），視覺聽覺交替衝擊，營造悲壯的氣氛。
- **徵人之口控訴**：借老百姓自己的話來反映戰爭的痛苦，使讀者更加感同身受。
- **意象鮮明**：「牽衣頓足攔道哭」七個字活現親人生離死別的悲痛場面。
- **寓情於事**：全詩感情蘊藏在客觀的記敍之中，杜甫沉鬱頓挫的詩風在詩中充分展現。

**（五）杜甫詩歌風格**

| 特點 | 說明 |
|------|------|
| 沉鬱頓挫 | 感情深沉鬱結，節奏抑揚頓挫，感染力強 |
| 寫實主義 | 善於捕捉現實社會問題，反映民間疾苦 |
| 語言凝鍊 | 用詞簡潔有力，意象鮮明 |
| 緣事而發 | 因有感於時事而創作，具有強烈的社會責任感 |


## 二、語文運用

### 單元三：肯定句、否定句

**（一）定義**

| 分類 | 解釋 | 例子 |
|------|------|------|
| 肯定句 | 表達事物「是這樣」，對事實作正面斷定的句子 | 「他是好學生。」、「這次考試他考得很好。」 |
| 否定句 | 表達事物「不是這樣」，對事實作反面斷定的句子，通常含有「不」、「沒有」等否定詞 | 「他不是壞人。」、「這次考試他考得不好。」 |

在中文裏，肯定句和否定句都屬於陳述句。陳述句的作用是敍述一件事實或表達說話者的看法。

**（二）類型**

- **一般否定句**：即含有「不、沒有、完全不是、完全不」等否定意思的詞語的句子。

**（三）句式轉換**

肯定句與否定句可以互相轉換，轉換的原則是：**不改變句子的原意**。

- **肯定句 → 否定句**：可按照「否定詞 + 原詞的反義詞」的方式進行。例如：「他考得很好」→「他考得不錯」

**（四）例子分析**

| 原句 | 類型 | 轉換 |
|------|------|------|
| 他喜歡讀書。 | 肯定句 | → 他不是不喜歡讀書。（雙重否定強調肯定） |
| 他沒有遲到。 | 否定句 | → 他準時上學。 |
| 文章寫得很動人。 | 肯定句 | → 文章寫得不差。 |


### 單元六：借代

**（一）定義**

借代是修辭學中的一種辭格，是指在談話或行文中，放棄通常所用的本名或語詞不用，而另找其他相關的名稱或語詞來代替。

- 被代替的事物稱為**「本體」**
- 用來代替的事物稱為**「借體」**
- 借體與本體之間必須有密切的相關性

**（二）常見分類**

| 分類 | 說明 | 例子 |
|------|------|------|
| 以事物的特徵或標誌代替事物 | 用該事物最明顯的特徵來替代 | 「怕嗎？小嫩皮。」 — 「小嫩皮」代未經世面的小孩 |
| 具體事物代替抽象概念 | 用具體可感的物品代替抽象的情感 | 「我為了明天的麪包而工作」 — 「麪包」代生活的物質基礎 |
| 部分代全體 | 用事物的某一部分代表整體 | 黃髮代老人，垂髫代小孩（源自《桃花源記》） |
| 以專名代通名 | 用著名的人或事物代替同類事物 | 「何以解憂，唯有杜康」 — 「杜康」（善造酒者）代「酒」 |

**（三）更多例子**

| 例子 | 出處／說明 | 借體 | 本體 |
|------|-----------|------|------|
| 臣本布衣，躬耕於南陽 | 諸葛亮〈前出師表〉 | 布衣 | 平民百姓 |
| 孤帆遠影碧山盡 | 李白詩句 | 帆 | 船 |
| 兵馬倥傯……半付祝融 | 連橫〈臺灣通史自序〉 | 祝融 | 火 |
| 宴酣之樂，非絲非竹 | 歐陽修〈醉翁亭記〉 | 絲竹 | 音樂 |
| 走向紅毯的另一端 | 現代用語 | 紅毯的另一端 | 婚姻（結婚） |

**（四）功用**

借代可以使語言表達新鮮活潑，使筆下事物的形象更深刻，富有俏皮感和生活氣息。


### 單元七：假設複句

**（一）定義**

假設複句又稱假設句，是指分句之間具有假設關係的複句。前邊的偏句提出一個假設的情況，後邊的正句說明在這種假設情況下會產生的結果。

**（二）常用關聯詞語**

| 常見關聯詞 | 例子 |
|------------|------|
| 如果……就…… | 如果你再說謊下去，就沒有人會再相信你了。 |
| 假如……那麼…… | 假如只是埋頭苦讀，不動腦筋想一想，那就得不到收穫。 |
| 要是……就…… | 要是你選擇了B，那就恭喜你答對了。 |
| 倘若……便…… | 倘若人品教養都不好，學識再豐富也是枉然的。 |
| 即使……也…… | 即使不是牽強附會，也是以犧牲詩的藝術為代價的。 |

**（三）結構分析**

假設複句由兩個有假設關係的分句組成：

> **前句（偏句）** ：假設一種情況  →  **後句（正句）** ：描述該情況下產生的結果

**例子分析**：
- 「**如果**你努力溫習，**就**能在考試中取得好成績。」
- 前句假設一個情況（努力溫習），後句說明結果（取得好成績）。
- 「如果……就……」是常見的關聯詞語。

**（四）假設複句的種類**

| 種類 | 特點 | 常用關聯詞 |
|------|------|------------|
| 一致假設複句 | 假設情況成立就會產生相應結果 | 如果……就……、假如……便…… |
| 讓步假設複句 | 先退一步承認情況成立，再説出不相應的結果 | 即使……也……、縱然……還…… |

**（五）例子練習**

| 句子 | 分析 |
|------|------|
| 如果你明天準時出席，我們就開始會議。 | 「如果……就……」假設情況：出席會議；結果：開始會議 |
| 即使天下大雨，我也會準時到達。 | 「即使……也……」假設最壞情況仍然堅持行動 |
| 倘若你肯虛心學習，必定能有所進步。 | 「倘若……必定……」假設與結果相應 |

> 📝 **考試提示**：
> - 假設複句中常出現的關聯詞包括：如果、假如、倘若、要是、即使等
> - 注意區分：「條件複句」側重於條件成立就產生結果（如「只要……就……」），而「假設複句」側重於「假設的情況」
