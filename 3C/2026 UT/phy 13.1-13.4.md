【 第14章 升級版筆記：光與光的現象（附深度解析與實戰秘笈） 】

［ 14.1 物體發出的光線 ］
1. 發光體 (Luminous object)：本身會發光的物體（如太陽、燈泡、火焰）。
2. 非發光體：本身不發光，靠反射光線被看見（如月亮、書本、桌子）。
   💡 重點解析：我們能看見物體，是因為光線進入了我們的眼睛。非發光體是把環境中的光「彈（反射）」進我們的眼睛裡。
3. 傳播方式：光沿「直線傳播」。
   - 光線 (Ray)：表示光傳播方向的線（畫圖時必須帶箭頭 ->）。
   - 光束 (Beam)：一束光線。

［ 14.2 反射 (Reflection) ］
1. 反射定律 (Laws of Reflection)：
   - 入射線、反射線和法線均在「同一平面」上。
   - 入射角 (i) = 反射角 (r)。
   🚨 必考陷阱：魔鬼細節「與表面夾角」
     角度永遠是和「法線（垂直於交界面的虛線）」的夾角，絕對不能量成和介質表面的夾角！
     如果題目說「光線與鏡面的夾角為 40 度」，真正的入射角 i = 90度 - 40度 = 50度。
2. 平面鏡成像性質：
   - 正立 (Erect)：上下不會顛倒。
   - 虛像 (Virtual)：光線看似從鏡後發出，不能投射在屏幕上。
   - 橫向倒置 (Laterally inverted)：左變右，右變左。
   - 像與物大小相等。
   - 像距 = 物距。
3. 應用：擴闊視野（凸面鏡）、潛望鏡。

［ 14.3 折射 (Refraction) ］
1. 定義：當光線由一種介質進入另一種介質時，因「光速改變」而導致傳播方向改變的現象。
2. 常用物質折射率 (n) 參考：（n 越大，光速越慢，介質越「光密」）
   - 空氣/真空 約為 1.00（基準點）
   - 水 約為 1.33
   - 有機玻璃/玻璃 約為 1.50
   - 鑽石 約為 2.42（折射率極大）
3. 偏折方向判斷：
   - 光疏 -> 光密（例：水 -> 鑽石）：光速減慢，光線「偏向法線」（入射角 > 折射角）。
   - 光密 -> 光疏（例：玻璃 -> 空氣）：光速加快，光線「偏離法線」（入射角 < 折射角）。
   🚨 特例：垂直入射 (i = 0度)：當光線沿著法線直直射入交界面時，光線「不會發生偏折」（直穿過去），但光速依然會改變！

★ 核心計算：司乃耳定律 (Snell's Law)
公式：n1 × sin(θ1) = n2 × sin(θ2)

🚨 實戰題型：多層平行介質（空氣 -> 玻璃 -> 水）
- 題目描述：光線以 50度 射入裝有水的玻璃缸。
- 解題秘笈：平行介質有一個特性：「最終的折射角，等同於光線直接從第一層射入最後一層」。中間的玻璃層雖然讓光線平移了，但不會改變最終射入水中的角度。
- 計算方式：你可以直接略過玻璃，算（空氣 -> 水）：1 × sin(50度) = 1.33 × sin(最終折射角)。

［ 14.4 全內反射 (Total Internal Reflection) ］
1. 發生條件（必須同時符合兩個，缺一不可）：
   - (i) 光線必須由「光密介質」射向「光疏介質」（例如：玻璃 -> 空氣，n大 -> n小）。
   - (ii) 入射角 (i) > 臨界角 (C)。
2. 應用：
   - 光纖（網路傳輸、內視鏡）、鑽石閃耀、路面貓眼石。

★ 核心計算：臨界角 (Critical Angle, C)
定義：當折射角剛好等於 90度（光線貼著交界面走）時的入射角。
萬用公式（光密 n1 射向 光疏 n2）：sin(C) = n2 / n1 
(記憶法：永遠是小的 n 除以大的 n)
如果射向空氣 (n 約為 1)，公式簡化為：sin(C) = 1 / n

🚨 實戰題型一：光纖 (Optical Fiber) 的雙層介質計算
- 題目描述：光纖的核心 (Core) n=1.62，外圍的包層 (Cladding) n=1.50。光線在核心內以 70度 射向包層交界面，會發生什麼事？
- 解題秘笈：
  1. 先算臨界角：光線從核心（光密）射向包層（光疏），使用萬用公式 sin(C) = 包層n / 核心n = 1.50 / 1.62。得出臨界角 C 約為 67.8度。
  2. 判斷現象：因為實際入射角 70度 > 臨界角 67.8度，符合全內反射條件，所以會發生「全內反射」，光線完美保留在核心內，不會漏到包層去。

🚨 實戰題型二：半圓形玻璃磚 (Semicircular block) 的綜合考驗
- 題目描述：一束光線由半圓形弧面「垂直」射入玻璃磚 (n=1.52)，直達平直邊緣的圓心 O 點。到達圓心時與平直邊緣法線夾角為 45度。試描述光線路徑。
- 解題秘笈（分為兩個階段看）：
  1. 弧面進入點（折射特例）：因為光線「垂直」射入弧面，代表入射角 i = 0度（沿著半徑走，剛好與法線重合）。此時光線「不會偏折」，直線穿過直到圓心 O 點。
  2. 平直邊緣（圓心 O 點出發）：
     - 此時光線從玻璃（光密）準備射向空氣（光疏），要立刻警覺全內反射！
     - 先算臨界角：sin(C) = 1 / 1.52，得出臨界角 C 約為 41.1度。
     - 判斷：題目說到達圓心時，與法線夾角（入射角）為 45度。因為 45度 > 41.1度 (i > C)，所以發生「全內反射」！
     - 結論：光線不會折射出去空氣，而是以 45度 的反射角（根據反射定律）直接反射回玻璃內部。
