## 一、 七大食物成分 (Seven Food Components)

### 1. 提供能量的成分 (Energy-providing components)

- **碳水化合物 (Carbohydrates)**:
    
    - **基本單位**: 單糖 (Monosaccharides)，如葡萄糖 (Glucose)。
        
    - **分類**: 單糖（葡萄糖）、雙糖 (Disaccharides，如蔗糖、麥芽糖)、多糖 (Polysaccharides，如澱粉、纖維素、糖原)。
        
    - **功能**: 主要能量來源（熱量值：17 kJ/g）。
        
- **脂質 (Lipids)**:
    
    - **基本單位**: 甘油 (Glycerol) + 脂肪酸 (Fatty acids)。
        
    - **功能**: 能量儲備、保暖（皮下脂肪）、保護內臟、構成細胞膜（熱量值：39 kJ/g）。
        
- **蛋白質 (Proteins)**:
    
    - **基本單位**: 氨基酸 (Amino acids)。
        
    - **功能**: 生長與修補細胞、製造酶 (Enzymes)、抗體 (Antibodies) 和激素 (Hormones)（熱量值：18 kJ/g）。
        
    - **注意**: 過量的氨基酸會在肝臟進行**脫氨作用 (Deamination)**，轉化成尿素 (Urea) 隨尿液排出。
        

### 2. 保護性及其他成分 (Protective & other components)

- **維生素 / 維他命 (Vitamins)**:
    
    - **維生素 A**: 脂溶性，與視覺有關（視紫素原料）。
        
    - **維生素 C**: 水溶性，維持齒齦及皮膚健康，不耐熱。
        
    - **維生素 D**: 脂溶性，促進鈣、磷吸收，鞏固骨骼。
        
- **礦物質 / 無機鹽 (Minerals)**:
    
    - **鈣 (Calcium)**: 構成骨骼及牙齒，與凝血有關。
        
    - **鐵 (Iron)**: 製造血紅蛋白 (Haemoglobin) 的主要成分。
        
    - **碘 (Iodine)**: 構成甲狀腺素。
        
- **食用纖維 (Dietary Fiber)**: 不能被消化吸收，但可刺激**腸蠕動 (Peristalsis)**，預防便秘 (Constipation)。
    
- **水分 (Water)**: 原生質 (Protoplasm) 主要成分、良好的溶劑、幫助運輸及調節體溫。
    

## 二、 營養缺乏病 (Deficiency Diseases)

| 缺乏物質          | 導致病症                      | 主要症狀        |
| ------------- | ------------------------- | ----------- |
| **蛋白質**       | 蛋白質缺乏症                    | 身體瘦弱、腹脹、易感染 |
| **維生素 A**     | **夜盲症 (Night blindness)** | 黑暗中視力模糊     |
| **維生素 C**     | **壞血病 (Scurvy)**          | 齒齦出血、傷口難癒合  |
| **維生素 D / 鈣** | **佝僂病 (Rickets) / 軟骨病**   | 骨骼變軟、雙腿彎曲   |
| **鐵質**        | **貧血 (Anemia)**           | 臉色蒼白、容易疲倦   |
| **碘質**        | **甲狀腺腫 (Goitre / 大頸泡)**   | 頸部腫脹        |

## 三、 食物試驗 (Food Tests)

- **澱粉測試 (Starch test)**: 使用**碘液 (Iodine solution)**。陽性反應：由棕色變**藍黑色**。
    
- **葡萄糖測試 (Glucose test)**: 使用**尿糖試紙 (Glucose test paper)**。
    
- **蛋白質測試 (Protein test)**: 使用**蛋白質試紙 (Protein test paper)**。
    
- **脂質測試 (Lipid test)**: **油漬試驗 (Grease spot test)**。在濾紙留下恆久不褪的**半透明點**。
    
- **維生素 C 測試 (Vitamin C test)**: 使用 **DCPIP 溶液**。陽性反應：由藍色變**無色**。
    
    - _量化實驗 (Quantitative experiment)_：比較不同果汁滴數，滴數愈少代表維生素 C 濃度愈高。
        

## 四、 消化系統 (Digestive System)

### 1. 機械消化 (Mechanical Digestion)

- **口腔**: 牙齒咀嚼。
    
    - **門齒 (Incisors)**: 切斷食物。
        
    - **犬齒 (Canines)**: 撕碎肉塊。
        
    - **前/臼齒 (Pre/Molars)**: 磨碎食物。
        
- **胃**: 肌肉攪動將食物變成糊狀 (Chyme)。
    
- **小腸**: **膽汁 (Bile)**（由肝製造，膽囊儲存）將脂質**乳化 (Emulsification)** 成微滴。
    

### 2. 化學消化 (Chemical Digestion)

- **口腔**: 唾液澱粉酶將澱粉分解成糖。
    
- **胃**: 胃液含**胃蛋白酶 (Pepsin)** 及**氫氯酸 (HCl)**（殺菌、提供酸性環境）。
    
- **小腸**: 胰液及腸液含多種酶，將碳水化合物、脂質、蛋白質徹底分解為單糖、脂肪酸、氨基酸。環境呈**鹼性**。
    

### 3. 吸收 (Absorption)

- **小腸絨毛 (Villi)**: 增加表面積。
    
    - **微血管**: 吸收水溶性營養（葡萄糖、氨基酸、礦物質、維生素 C）。
        
    - **乳糜管 (Lacteal)**: 吸收脂溶性營養（脂肪微滴、維生素 A/D）。
        

## 五、 血液循環系統 (Circulatory System)

### 1. 血管種類比較 (Blood Vessels)

- **動脈 (Artery)**: 運血離心。管壁厚、彈性強、血壓高、無瓣膜。
    
- **靜脈 (Vein)**: 運血回心。管壁薄、彈性弱、血壓低、**有瓣膜**（防止倒流）。
    
- **微血管 (Capillary)**: 物質交換。管壁極薄（單層細胞）、管腔窄、血流慢。
    

### 2. 心臟與雙循環 (Heart & Double Circulation)

- **心臟腔室**: 左心室 (L.V.) 壁最厚（需泵血往全身）。
    
- **循環路徑**:
    
    - **肺循環 (Pulmonary circulation)**: 右心室 → 肺動脈 → 肺 → 肺靜脈 → 左心房。
        
    - **體循環 (Systemic circulation)**: 左心室 → 主動脈 → 全身 → 大靜脈 → 右心房。
        
- **含氧血 vs 缺氧血**: 動脈一般運送含氧血（肺動脈例外）；靜脈一般運送缺氧血（肺靜脈例外）。
